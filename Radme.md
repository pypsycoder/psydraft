# Psydraft

Конструктор психиатрического заключения — интерактивная HTML-форма для быстрого ввода психостатуса и формирования развернутого заключения.

## Ветки

- **main** — стабильная версия
- **dev** — разработка, изменения, интеграция с Codex Web

## Структура проекта

psydraft/
│
├── html/
│     └── шаблон_консультации_психиатра.html
│
├── LICENSE
├── README.md
└── .gitignore

## Как использовать

1. Открыть файл `html/шаблон_консультации_психиатра.html` в браузере  
2. Заполнить поля  
3. Нажать «Сформировать текст»  
4. Скопировать результат и вставить в EMR/Word

## Разработка

Все изменения вносятся через ветку `dev`.  
После тестирования создаётся pull request в `main`.
